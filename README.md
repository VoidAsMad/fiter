# Fiter
[<img src="https://img.shields.io/pypi/v/fiter.svg">](https://pypi.python.org/pypi/fiter)<br>
## Installation
```
$ pip install fiter
```
### How to Use
```py
from fiter import number

fiter = number.Conversion()
print(fiter.format('5000000000', format = '코인'))
```
#### Response
```
50억 코인
```
